This file is to explain how to run the program.

In order to run the program:
	1. Compile and run Gui.java
	2. Click which case you would like to run (base or optimized)
	3. Enter a number between 0 and Integer_MAX
	4. Click run
	5. Wait for results.

In order to run for testing:
	1. Compile Testing.java
	2. Change the variable tests to maxTrains / 10;
	3. Run program
	4. Train-numTrains-case.txt and TrainData-numTrains-case.csv files have been created for each test
